Keys that differ from modern keyboard markings. 
F9 selects Key Map 1
F10 selects Key Map 2 (Scroll Lock LED will lite)

Keymap 1 Symbolic

C64	Modern
@	~ (Tide)
c=	Windows (Graphic Key)
RUN	ALT
Pound	ESC
:	' (Quote Key)
L ARR	Left Cursor Key
UP ARR	Up Cursor Key
HORZ	Right Cursor Key
VERT	Down Cursor Key
Restore	Tab

Keypad Active keys: + - * / Enter

Keymap 2 Positional
C64	Modern
+	-
-	=
@	[
*	]
;	' (Quote Key)
Run	Ctrl
C=	ALT/Win
CTRL	Tab
L ARR	~ Tide
Pound	Backspace
=	END
Keypad Active: + _ *
